## Bot Files

| File | Description |
|------|-------------|
| `telegram_bot.py` | Telegram chatbot — courses, booking, leads |
| `whatsapp_bot.py` | WhatsApp bot via Twilio — full menu system |

### Setup Bots

```bash
# Telegram
pip install python-telegram-bot
# Add your token from @BotFather to telegram_bot.py
python telegram_bot.py

# WhatsApp
pip install flask twilio
# Add Twilio credentials to whatsapp_bot.py
python whatsapp_bot.py
# Then run: ngrok http 5000
```
